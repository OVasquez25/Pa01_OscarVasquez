Introducción al Desarrollo Web

Este es un ejemplo práctico para demostrar los conocimientos adquiridos en el curso de Diseño Web, enfocándonos en HTML5 para la estructura y CSS3 para el estilo.

El desarrollo web es un campo fascinante que nos permite construir experiencias interactivas y visualmente atractivas en línea. Se divide principalmente en dos grandes áreas:

  1. **Frontend:** Lo que el usuario ve e interactúa directamente en el navegador (HTML, CSS, JavaScript).
  2. **Backend:** La lógica del servidor, bases de datos y la gestión de la información (lenguajes como Python, PHP, Node.js, etc.).

Explorando HTML Semántico

El Poder de la Semántica en HTML5

HTML5 introdujo un conjunto de etiquetas semánticas que ayudan a los navegadores y motores de búsqueda a entender el significado de las diferentes partes de una página. Esto no solo mejora la **accesibilidad** sino también el **SEO**.

Elementos Comunes de HTML5:

  • `<header>`: Para el encabezado de un documento o sección.
  • `<nav>`: Para la navegación principal.
  • `<main>`: Contenido principal y único de la página.
  • `<article>`: Contenido independiente y reutilizable.
  • `<section>`: Temas relacionados.
  • `<aside>`: Contenido tangencial o barras laterales.
  • `<footer>`: Pie de página del documento o sección.

Utilizar estas etiquetas de forma adecuada es una buena práctica y fundamental para un desarrollo web moderno. Por ejemplo, un enlace a otra sección de esta misma página: Volver a Introducción.

Estilizando con CSS3

CSS3 nos permite dar vida a nuestros diseños, controlando colores, tipografías, espaciados y disposición de los elementos. Es el lenguaje del estilo en la web.

Modelo de Caja y Propiedades CSS

Cada elemento HTML es una "caja" con propiedades como:

  • **`margin`**: Espacio exterior al borde.
  • **`border`**: Línea alrededor del elemento.
  • **`padding`**: Espacio interior entre el contenido y el borde.
  • **`content`**: El contenido real del elemento.

Utilizamos selectores para apuntar a elementos específicos, como el selector de clase `.caja-ejemplo` o el selector de tipo `p`. También podemos usar unidades de medida como `px`, `em`, `%`, y codificación de colores como `#RRGGBB` o `rgb()`.
